About

Golf Mental Form Analysis System

This system analyzes the mental state of professional golfers by extracting qualitative insights from podcasts, interviews, and other sources. It uses these insights to calculate a mental form score that can help predict how a player might perform relative to their statistical expectations.

How It Works

  1. Player data is imported from the Data Golf API
  2. Transcripts are processed to extract qualitative insights about players
  3. The Claude API analyzes these insights to calculate mental form scores
  4. Scores are tracked over time to identify trends and patterns

Mental Form Score Scale

  • -1.0 to -0.3: Severely compromised mental state
  • -0.3 to 0.3: Neutral mental state
  • 0.3 to 1.0: Exceptional mental state

Components

  • Database: SQLite database storing player information, insights, and mental form scores
  • Claude API: Used for extracting insights and calculating mental form scores
  • Data Golf API: Source of player data and statistics
  • Flask Web Interface: This web application for managing the system
